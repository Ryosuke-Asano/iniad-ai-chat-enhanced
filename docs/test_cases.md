# 手動テスト観点一覧 (INIAD AI Chat Enhanced)

本ドキュメントは、タスク A11 に基づき、Renderer プロセス（フロントエンド）の主要機能を確認するための手動テスト項目をまとめたものである。

## 1. チャット画面 (A01, A02, A03)

| ID | テスト項目 | 確認内容 | 期待結果 |
| :--- | :--- | :--- | :--- |
| CH01 | メッセージ送信 | テキストを入力し、送信ボタン（またはEnter）を押下する | 入力したメッセージが右側（User）に表示される |
| CH02 | AI応答待機 | 送信直後の状態を確認する | 送信ボタンと入力欄が disabled になり、Loading状態（StatusBar等）が表示される |
| CH03 | AI応答表示 | ダミーのAI応答が返ってきた際の表示を確認する | メッセージが左側（Assistant）に表示され、アイコンと名前が正しく表示される |
| CH04 | 改行保持 | 複数行のメッセージを送信する | `pre-wrap` により改行が正しく保持されて表示される |
| CH05 | 空送信防止 | 何も入力せずに送信を試みる | 送信ボタンが非活性であるか、送信が実行されないこと |

## 2. 参照元（Citation）パネル (A04)

| ID | テスト項目 | 確認内容 | 期待結果 |
| :--- | :--- | :--- | :--- |
| CI01 | パネル表示 | AI応答に `citations` データが含まれる場合を確認する | 回答文の下に「参照元」セクションが表示される |
| CI02 | リンク遷移 | 参照元のURLをクリックする | ブラウザで正しいURLが開かれる（または Electron の shell.openExternal で開く動作） |
| CI03 | パネル非表示 | `citations` が空の場合を確認する | 「参照元」セクションが表示されないこと |

## 3. ステータスバー (A05, A06)

| ID | テスト項目 | 確認内容 | 期待結果 |
| :--- | :--- | :--- | :--- |
| ST01 | MCP接続状態表示 | 接続状態に応じたインジケータの色を確認する | connected: 緑, connecting: 黄, disconnected: 赤 |
| ST02 | モデル名表示 | 現在のモデル名が表示されているか確認する | 指定されたモデル名が正しく表示される |

## 4. 設定画面 (A07, A08)

| ID | テスト項目 | 確認内容 | 期待結果 |
| :--- | :--- | :--- | :--- |
| SE01 | APIキー入力・保存 | 新しいAPIキーを入力し「保存」する | 保存成功メッセージが表示され、画面を閉じても保持される |
| SE02 | シークレット入力 (Peek) | APIキー/パスワード入力時に1文字打つ | 打った文字が一瞬表示され、800ms後に ● に変わる |
| SE03 | 表示/非表示切替 | 「目」のアイコンを長押し（またはクリック）する | 押している間だけ平文で表示される |
| SE04 | バリデーション | 不正なURLや空の必須項目で保存を試みる | エラーメッセージが表示され、保存が阻止される |
| SE05 | 接続テスト | 各テストボタンをクリックする | 実行中にローディングアニメーションが表示され、成否がアイコンで通知される |

## 5. 画面切替トランジション (A09)

| ID | テスト項目 | 確認内容 | 期待結果 |
| :--- | :--- | :--- | :--- |
| TR01 | ビュー切替 | 設定ボタン / 閉じるボタンを押下する | チャット画面と設定画面が滑らかに切り替わる（フェード/スライド） |
| TR02 | 状態保持 | 設定画面からチャット画面に戻る | 入力中だったメッセージや、これまでの会話履歴が消えずに保持されている |

## 6. レスポンス・ポリッシュ (A10)

| ID | テスト項目 | 確認内容 | 期待結果 |
| :--- | :--- | :--- | :--- |
| PO01 | レスポンス対応 | ウィンドウ幅を極端に狭く・広くする | 各要素が重なったり画面外にはみ出したりせず、適切に配置される |
| PO02 | スクロール動作 | 長い会話履歴を表示する | 最新のメッセージが常に（または手動スクロールで）確認でき、ヘッダー/入力欄が固定されている |
